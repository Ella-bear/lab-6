# lab-6

## Run Locally
Clone the project

```bash
  git clone https://github.com/PLP-Database-DEPT/lab-6.git
```

Go to the project directory

```bash
  cd lab-6
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Import users data

```bash
  py main.py
```
Import post data

```bash
  py post.py
```
